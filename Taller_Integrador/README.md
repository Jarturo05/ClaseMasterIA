# Ejecución del Notebook

Para la ejecución del notebook se requiere 
1. Instalar Visual Studio Code
2. Tener la version 3.11.5 de Python (Para evitar cualquier problema con actualizaciones futuras o antiguas)
3. Tener la extension llamada Jupyter
4. Una vez instalado dichas instalaciones simplemente con ejecutar el archivo taller_modulo1.ipynb se realizara la respectiva ejecución del mismo
5. Nota: Se añadio %pip install pandas dentro del archivo para que al ejecutrar con Run All se instale automaticamente y ejecute los puntos del DataFrame (df)
